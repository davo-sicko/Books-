# BooksOOP

Консольное приложение на Java для управления библиотекой книг.

## Функции

- Добавление книги
- Просмотр всех книг
- Поиск книги по ID
- Изменение названия книги
- Удаление книги по ID
- Автоматическая генерация ID
- Работа через консольное меню

## Используемые технологии

- Java
- ООП (классы и объекты)
- ArrayList
- Методы
- Сканер для ввода данных

## Реализованные возможности

- Создание объектов класса Book
- Хранение объектов в коллекции ArrayList
- CRUD-операции (Create, Read, Update, Delete)
- Проверка корректности пользовательского ввода
- Поиск объектов по идентификатору

## Цель проекта

Проект создан в рамках подготовки к экзамену по дисциплине 
«Основы алгоритмизации и программирования (Java)».
